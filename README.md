# Glassmorphism ATM System

A modern ATM system interface featuring **glassmorphism design** for a sleek and futuristic look. This project demonstrates a simple ATM system with basic functionalities like checking balance, withdrawing cash, and depositing money, combined with an attractive glassmorphism UI.

## Features

- Glassmorphism-inspired user interface
- Check balance, deposit, and withdraw functionalities
- Responsive design for different screen sizes
- Simple and easy-to-use ATM simulation
